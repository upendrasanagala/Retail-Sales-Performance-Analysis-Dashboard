# Retail-Sales-Performance-Analysis-Dashboard

#### Built an end-to-end sales analytics project using Excel, SQL Server, Python, and Power BI
#### Cleaned and prepared 51K+ transaction records using Excel
#### Performed business analysis using SQL aggregation queries on regional performance, category profitability, and customer segmentation
#### Conducted exploratory data analysis using Python (pandas, matplotlib, seaborn)
#### Developed an interactive Power BI dashboard with KPI cards, trend analysis, and dynamic filters
#### Identified loss-making sub-categories and regional revenue drivers through visualization insights

## Tools Used
- Excel (Data Cleaning)
- SQL Server (Business Queries)
- Python (EDA & Visualization)
- Power BI (Interactive Dashboard)
